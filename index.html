Scfsi

Un outil de recherche et d'indexation dans les fichiers binaires à base de mots clés

Bienvenue à la page de notre outil SCFSI.

Nous vous présentons un script Médico-légal de recherche et d'indexation de preuves à base de mots clés en entrée dans un fichier biniare en examen. Le programme est constitué de deux fichiers Python, SCFSI.py et SCFSI_Recherche.py.

Avant de commencer

Le script à exécuter en ligne de commande est SCFSI.py Les deux fichiers SCFSI.py et SCFSI_Recherche.py doivent être dans le même répertoire. Un fichier .txt de mots clés doit être fourni en entrée au programme. Un fichier binaire sur lequel doit s'appliquer le script ( exemple un fichier de capture réseau) Un fichier .txt des résultats en sortie doit être mensionné.

Comment l'utiliser

Sur un terminal ou un prompt shel cmd.exe, exécutez le script comme suit:

syscore$ python SCFSI.py -k mostclefsàchercher.txt -t capture.raw > resultats.txt

Indexation

L'indexation offre à l'enquêteur différentes vues sur les données et de potentielles preuves contenues dans un fichier, image de disque, capture de mémoire ou une trace réseau. Sur un terminal ou un prompt shel cmd.exe, exécutez le script comme suit:

syscore$ python SCFSI.py -k mostclefsàchercher.txt -t capture.raw -m matrice.txt
